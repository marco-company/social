Usage
=====

Blacklisted contacts can be removed while creating a new mailing:

1. Create a new mailing
2. Select a mailing list that contains blacklisted contacts: a button "Prune lists..." appears
3. Click on "Prune lists...": a wizard appears
4. In the wizard, the blacklisted contacts are shown
5. Confirm: the blacklisted contacts are removed from the selected lists

The same wizard is also among the available actions for mailing lists.
